# PRODUCT.md — CV Verdict

## Product
CV Verdict is a bilingual Arabic/English public website that evaluates CVs with strict recruiter-style logic.

## Mission
Give job seekers a realistic, evidence-based CV score from 100, explain why they may be rejected, and show what to fix first.

## Audience
- Students.
- Fresh graduates.
- Internship applicants.
- Entry-level applicants.
- Mid-level applicants.
- Arabic and English users.

## Core flow
1. User uploads PDF CV.
2. User chooses report language.
3. User may add target role.
4. User may add job description.
5. User selects experience level.
6. User selects free report depth: short or medium.
7. System analyzes CV.
8. User receives score, decision, strengths, risks, ATS summary, and repair actions.

## Differentiation
- Strict, not flattering.
- Evidence-based.
- Credibility-aware.
- Looks for recruiter objections.
- Does not treat skill lists as proof.
- Does not invent missing numbers.
- Almost-free / low-cost.
- Bilingual.
- Distinctive frontend, not a generic SaaS template.

## MVP boundaries
In:
- PDF upload.
- Text extraction.
- Structured parsing.
- Basic CV evaluation.
- Arabic/English report.
- Short/medium free reports.
- Delete data action.

Out:
- Accounts.
- Payments.
- OCR.
- Word upload.
- LinkedIn import.
- CV editor.
- Admin dashboard.
- Bulk evaluation.
