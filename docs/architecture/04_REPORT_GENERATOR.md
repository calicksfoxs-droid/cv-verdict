# 04 — Report Generator v1.0

## Purpose
Convert Evaluation Result JSON into a user-readable report without changing the score.

## Forbidden
Report Generator must not:
- Change scores.
- Add new claims.
- Invent achievements.
- Add fake keywords.
- Hide critical issues.
- Change issue severity.

Allowed:
- Reorder.
- Summarize.
- Explain.
- Format.
- Translate between Arabic and English where needed.

## Full report sections
1. Executive judgment.
2. First 10-second test.
3. Final score.
4. Score breakdown.
5. Top five strengths.
6. Top ten issues.
7. Section-by-section analysis.
8. ATS analysis.
9. Credibility review.
10. Rewrite weakest points.
11. Professional summary review.
12. Questions for the candidate.
13. Repair plan.
14. Expected score after improvement.
15. Final decision.

## Report quality checks
- Score matches Evaluation Result.
- No invented info.
- No duplicate issue spam.
- Arabic and English are natural.
- Final decision is one of the four approved options.
- Bullet rewrites do not invent metrics.
