# 01 — Product Scope v1.0

## Goal
Build a public website for strict CV evaluation. The site evaluates the CV document as a recruiter would see it, not the person behind it.

## Core promise
The user receives a realistic score from 100 and understands:
- Why the CV received that score.
- What would cause rejection.
- What is strongest.
- What is unsupported or unclear.
- What to fix first.

## Users
MVP target users:
- Students.
- Fresh graduates.
- Internship applicants.
- Entry-level and mid-level applicants.
- People applying in Arabic or English contexts.

Not MVP target:
- HR departments.
- Bulk recruitment teams.
- Enterprise ATS integrations.

## Input
Required:
- PDF CV.
- Experience level.

Optional:
- Target role.
- Job description.
- Target country.
- Opportunity type.
- Report language: Arabic or English.
- Report depth: free short or free medium.

Target role is optional by product choice. If absent, the system runs a generic evaluation and explicitly warns that job-fit scoring is limited.

## Output
MVP output:
- Final score from 100.
- Classification.
- Application decision.
- Executive judgment.
- 10-second review.
- Main score breakdown.
- Strongest points.
- Top issues.
- ATS summary.
- Repair steps.

Full output later:
- Full 14-section report.
- Credibility review.
- Bullet rewrites.
- Section-by-section review.
- Score forecast.

## Differentiation
This product differs from generic ATS checkers by being:
- Strict, not flattering.
- Evidence-based.
- Credibility-aware.
- Focused on recruiter objections.
- Honest about missing proof.
- Almost-free / low-cost by design.
- Bilingual Arabic/English.
- Visually distinctive, not a SaaS template.

## MVP exclusions
- Accounts.
- Payments.
- OCR.
- CV editing.
- CV generation.
- Word files.
- Persistent storage.
- LinkedIn import.
- Job search.
- Bulk CV review.
